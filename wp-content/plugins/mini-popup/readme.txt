== Description ==

[ENGLISH ~ INGL&Ecirc;S]

This plugin has the functionality to show a popup where the readers will see a fan box of the Facebook. This will encourage your readers to like your fan page and will generate more fans to your blog.

Some features of the plugin:

* Completely free.
* Very installation simple.
* Smart display mode, which increases the conversion and the satisfaction of the visitors.
* Simple, clean and light code.
* Cookies to improve the user experience.
* Translated to portuguese and english.
* Customizable close button.
* Compatible with older browsers too.

[ENGLISH ~ INGL&Ecirc;S]

* Go to the Settings menu and click on &quot;Mini Popup&quot;.
* Configure the plugin as you prefer following the page informations.
* Enjoy and don't forget to send us a review! =)

== Frequently Asked Questions ==

[ENGLISH ~ INGL&Ecirc;S]
= I installed and configured the plugin, but the plugin doesn't appear. = 
 * Check if you had not opened your blog and the cookie was recorded in the browser without you having seen.
 * Check if your theme has the functions wp_head in header.php, and wp_footer in footer.php.
 * Check if your theme is not loading the jQuery manually, through a line of code in header.php. JQuery conflicts may prevent the execution of the plugin on the page.
 * Clear your cache, libraries can not be loaded because was recorded a cache in the browser. You can test the plugin in other browsers or clear the cache of your wordpress blog.
 
= I installed the plugin, but many features of my theme stopped working and the popup doesn't appear. =
 * Check if your theme is not loading the jQuery manually, through a line of code in header.php. JQuery conflicts may prevent the execution of the plugin on the page.

= How is the compatibility of the plugin with responsive themes? = 
 * We use a line of code in the style plugin that hides the popup when the plugin runs on devices or monitors with a resolution of less than 480 pixels wide.
